# 비교 대시보드 (Comparison Dashboard)

> **상태: TBD** — 용어와 방향만 정의. 구체적 설계는 후순위.

## 개요

워크아웃 데이터를 다양한 기준으로 비교/분석하는 대시보드 기능.

## 비교 유형

### 1. 동반 러너 비교 (Companion Comparison)

- 같이 뛴 사람과의 기록 비교
- 매칭 기준: 같은 날짜 + 유사 GPS 위치 + 유사 시간대
- 비교 항목: 거리, 페이스, 심박 등

### 2. 개인 히스토리 비교 (Personal History Comparison)

- 과거 내 워크아웃과 현재 기록 비교
- 동일/유사 코스에서의 성장 추이
- 기간별 통계 (월간, 분기, 연간)

### 3. GPS 루트 비교 (Route Comparison)

- 비슷한 위치/코스를 뛴 데이터 자동 그룹핑
- 동일 코스 반복 시 기록 변화 추적
- 루트 유사도 알고리즘 필요 (TBD)
