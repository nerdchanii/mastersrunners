# 유저 프로필 (User Profile)

## 프로필 구성

### 기본 정보

| 필드 | 타입 | 설명 |
|------|------|------|
| 프로필 이미지 | image | 유저 프로필 사진 |
| 닉네임 | string | 표시 이름 |
| 소개글 | string? | 자기 소개 |
| 러닝 경력 | string? | 러닝 시작 시기/경력 |
| 목표 | string? | 러닝 목표 (예: "서브3", "매일 5km") |
| PB (개인 기록) | object? | 주요 거리별 최고 기록 |
| 계정 공개 설정 | enum | PUBLIC / PRIVATE |
| 단위 설정 | enum | KM / MI |

### 크루 소속

- 소속 크루 목록 표시
- 여러 크루 동시 소속 가능

## 프로필 탭

| 탭 | 내용 |
|----|------|
| Workouts | 본인 워크아웃 목록 (운동 기록) |
| Posts | 본인 포스트 목록 (소셜 콘텐츠) |
| Shoes | 등록된 신발 목록 |
| Race Records | 대회 공식 기록 목록 |

## 표시 제어

- 모든 프로필 표시 항목은 유저가 ON/OFF 설정 가능
- 예: PB 공개/비공개, 크루 소속 표시/숨김, 각 탭 노출 여부
- 비공개 계정은 팔로워만 프로필 상세 열람 가능

## 알림 (Notification)

### 알림 유형

**소셜**
- 새 팔로워 / 팔로우 요청
- 포스트에 좋아요
- 포스트에 댓글 / 대댓글
- 멘션 (@)

**크루**
- 가입 승인/거절
- 새 활동 생성
- 공지 등록
- 추방 알림

**대회**
- 참가 등록 확인
- 대회 일정 리마인더

**챌린지**
- 챌린지 초대 / 참여 승인
- 목표 달성 축하
- 챌린지 시작/종료

### 알림 설정

- 유형별 ON/OFF 설정 가능
- 푸시 알림 / 인앱 알림 구분 (구체적 구현은 TBD)
