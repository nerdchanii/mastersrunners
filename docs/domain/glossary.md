# 도메인 용어 사전

## 핵심 엔티티

| 용어 | 영문 | 설명 |
|------|------|------|
| 워크아웃 | Workout | 순수 운동 데이터 (거리, 페이스, 심박, 랩, 루트). 소셜 콘텐츠가 아닌 기록 단위 |
| 포스트 | Post | 소셜 콘텐츠 단위. 텍스트, 이미지, 해시태그, 워크아웃 첨부(0~N개) 가능 |
| 피드 | Feed | 포스트 목록 화면. 워크아웃 자체는 피드에 노출되지 않음 |
| 크루 | Crew | 러닝 동호회/클럽. 멤버, 활동, 출석, 공지 관리 |
| 챌린지 | Challenge | 목표 달성 이벤트. 유저/크루/플랫폼이 생성 가능 |
| 대회 | Event | 공식 마라톤 대회. 여러 종목(EventCategory)을 포함 |
| 종목 | EventCategory | 대회 내 개별 종목 (풀코스, 하프, 10km 등) |
| 신발 모델 | ShoeModel | 공유 제품 정보 (브랜드, 모델명, 카테고리) |
| 내 신발 | UserShoe | 유저가 소유한 개별 신발 인스턴스 |
| 크루 태그 | CrewTag | 크루 멤버에 부여하는 태그/호칭. 소그룹 역할 겸용 |

## 운동 관련 용어

| 용어 | 설명 |
|------|------|
| 랩 (Lap) | 워크아웃 내 구간 데이터. 모든 워크아웃 유형에서 표시 |
| 루트 (Route) | GPS 기반 경로 데이터 (위도, 경도, 고도, 타임스탬프) |
| 페이스 (Pace) | 단위 거리당 소요 시간 (예: 5:30/km) |
| 스플릿 (Split) | 일정 거리(보통 1km)마다의 구간 기록 |
| FIT 파일 | Garmin 등 GPS 워치의 표준 운동 데이터 파일 |
| GPX 파일 | GPS Exchange Format. 범용 GPS 경로 파일 |

## 소셜 용어

| 용어 | 설명 |
|------|------|
| 팔로우 (Follow) | 다른 유저의 포스트를 구독. 공개 계정은 즉시, 비공개 계정은 승인 필요 |
| 차단 (Block) | 상대방의 모든 접근 차단. 양방향 팔로우 해제 |
| 멘션 (Mention) | 댓글/포스트에서 @유저이름으로 상대방 언급. 팔로우 리스트 기반 자동완성 |
| DM (Direct Message) | 1:1 개인 메시지. 프로필에서 탭으로 대화 시작 가능 |
| 공개 범위 (Visibility) | PRIVATE / FOLLOWERS / PUBLIC. 기본값: FOLLOWERS |

## 크루 용어

| 용어 | 설명 |
|------|------|
| 크루 활동 (CrewActivity) | 정기런, 번개런 등 크루 단위 활동 |
| 출석 (Attendance) | QR 코드 기반 활동 참석 확인 |
| 추방 (Ban) | 크루에서 강제 퇴장. 해제 전까지 재가입 불가 |

## 대회 용어

| 용어 | 설명 |
|------|------|
| 공식 기록 (OfficialResult) | 대회 공식 성적 — gun time, net time, 순위, 에이지그룹 순위 |
| Gun Time | 출발 신호 기준 완주 시간 |
| Net Time | 출발선 통과 기준 완주 시간 (칩 타임) |

## 챌린지 용어

| 용어 | 설명 |
|------|------|
| 참여 단위 (Participation Unit) | INDIVIDUAL (개인) 또는 CREW (크루 합산) |
| 참여 방식 (Join Type) | OPEN (자동승인) 또는 APPROVAL (허가승인) |
| 챌린지 팀 (ChallengeTeam) | 챌린지 내부 팀 편성 (팀 모드 시) |
